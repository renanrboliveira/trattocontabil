# Termo de participação — piloto fundador

> **Rascunho.** Revisar com assessoria jurídica antes da assinatura.
> **Piloto inicial:** E2 only (5–10 clientes). E3/E4 entram após E2 fechar 1 competência.

## Partes

- **Prestador:** [razão social Extrato Pronto], doravante "Prestador"
- **Participante:** [razão social escritório], doravante "Escritório"

## Objeto

Participação gratuita no piloto da ferramenta Extrato Pronto, cobrindo o fluxo de recepção, conversão e exportação de extratos bancários para clientes indicados pelo Escritório.

## Prazo

60 a 90 dias a partir da data de kick-off: **2026-06-23**

## Escopo do piloto

- Até **10 clientes (CNPJs)** indicados pelo Escritório
- Sistema contábil alvo: **Alterdata** (confirmar versão: _______________)
- Canais: WhatsApp + e-mail

## Contrapartidas do Escritório

1. **Feedback quinzenal** de 20 min (roteiro em `docs/piloto/roteiro-feedback-quinzenal.md`)
2. **Extratos anonimizados** para calibração da conversão (conforme `docs/piloto/protocolo-lgpd-piloto.md`)
3. **Baseline de tempo** na semana 0 para os clientes piloto
4. **Opt-in** dos clientes finais para envio de extratos via canal digital
5. Permissão para uso de resultados como **caso anônimo** (se preferir)

## Contrapartidas do Prestador

1. Acesso gratuito à ferramenta durante o piloto
2. Suporte direto durante o período
3. Ao final, direito a **preço fundador vitalício (50% da tabela) até 30 CNPJs** contratados no piloto, se o Escritório optar por continuar

## Dados e LGPD

- Escritório = **controlador** dos dados dos clientes finais
- Prestador = **operador**, processando conforme instruções do Escritório
- Detalhes em `docs/piloto/protocolo-lgpd-piloto.md`

## Limitações

- Ferramenta em desenvolvimento; sem SLA de disponibilidade no piloto
- Escopo **exclui** funcionalidades fiscais (XML, SPED, apuração)
- Prestador não se responsabiliza por erros de lançamento não revisados pelo Escritório

## Encerramento

Qualquer parte pode encerrar com **7 dias de aviso**. Dados do Escritório excluídos em até 90 dias, salvo eval set já anonimizado.
