## Durante a call

### 1. O que funcionou / quebrou (5 min)

**Ela lista — anotar verbatim.**

### 4. Próxima prioridade (5 min)

"Pensando nas próximas duas semanas, o que mais te destravaria?"

**Ela escolhe** — vira input direto do backlog.
